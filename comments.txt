В проекте реализованы:
- создание схемы cafe, enum restaurant_type и таблиц;
- заполнение таблиц данными из raw_data;
- представление для топ-3 заведений по среднему чеку внутри каждого типа;
- материализованное представление для анализа изменения среднего чека по годам без 2023 года;
- аналитические запросы для задач 3–5;
- транзакционные обновления с блокировками для задач 6 и 7.

В task_6.sql используется блокировка строк через FOR UPDATE, чтобы заблокировать только заведения с капучино и не ограничивать доступ к остальным меню.

В task_7.sql используется LOCK TABLE cafe.managers IN SHARE MODE, чтобы таблица была доступна для чтения, но недоступна для изменений во время обновления телефонов.
